# Определение столкновений прямоугольников #
Этот проект реализует функции для работы с прямоугольниками:

1. Проверка корректности прямоугольников
2. Определение столкновений между прямоугольниками
3. Вычисление площади пересечения прямоугольников
4. Вычисление общей площади пересечения нескольких прямоугольников
# Использование #
Запустите 'main.py' для демонстрации всех функций (в bash): python3 main.py
